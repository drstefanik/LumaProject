# LUMA Speaking Project

Prototype of **LUMA** (Language Understanding Mastery Assistant),
a voice-based English speaking examiner for British Institutes.

## Setup

1. Copy `.env.local.example` to `.env.local` and fill in your keys.
2. Install dependencies:

   ```bash
   npm install
   ```

3. Run in dev:

   ```bash
   npm run dev
   ```

Open http://localhost:3000/luma to run a speaking test and
http://localhost:3000/admin/luma to view saved reports from Airtable.
